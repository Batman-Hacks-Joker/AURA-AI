# **App Name**: KARMA

## Core Features:

- ONBOARDING and LOGIN: First, determine if the user is a Customer or Admin. AI Guided steps will assist in store opening with providing at least one product detail for Admins. Customers will see available marketplaces. Login will authenticate users (customers/subadmins/admins) and authorize access to their respective areas.
- Admin Shop Creation: AI-guided shop creation (choose to open store, service center, warehouse) with smart prompting for product details. A blank icon will be used if no image is provided. The AI tool intelligently fills in missing details and creates initial product listings, enhancing admin efficiency and reducing manual data entry.
- Inventory Management for warehouses: Comprehensive inventory management for warehouses, including product/part details, accessories, customisations and available tools. If an admin has created a shop, they will by default have its warehouse section accessible for that product. Creation and management of warehouses in different locations. Admins can assign subadmins for warehouse or service center, subadmins are basically low level admins access within sections only.
- Service Center Management: Centralised platform for managing multiple service centers (if exists), including customer support dashboards and after-sales service tools. Admin can create a service center for its product by having a parent name and it might have product name but both of them are available in store/warehouse/service centers if admin has created both and allowed it be visible in these sections.
- Knowledge Base Creation: Sub-admins of service centers only can create and manage user manuals/FAQs for listed products, aiding in self-service support.
- Call Monitoring: Sub-admins can monitor inbound call summaries and track the progress of query resolutions handled by voice agents.
- User Product Dashboard: Verified customers access a unique product dashboard with details, vital parameters, customisation options, service appointment scheduling, and smart chatbot assistance via text, audio, or image queries for comprehensive product support and management. Users can login to see its purchased products, wish to buy from marketplace, or access the service support for the bought product. 
- Smart AI Service chatbot: Smart AI Service chatbot will be helping customers for their product hunt, product service support assisting by providing progress tracks of service requested (if exists). Admin can access KARMA Bot in Opening Store for their product, managing warehouses or analysing the service center performance with dashboards. User can input text, images or voices to interact with KARMA BOT

## Style Guidelines:

- Primary color: Emerald Green `#009B77` to convey sophistication and integration.
- Background color: Very light violet (#F0E5F5). Offers a subtle backdrop that keeps the focus on the content.
- Accent color: Deep Forest Green `#006B54`. This darker, analogous hue is used for emphasis on interactive elements such as buttons or links, creating a clear visual hierarchy.
- Body and headline font: 'Ranade' for a clean, modern sans-serif.
- Code font: 'Source Code Pro' for displaying code snippets.
- Consistent use of simple, clear icons across the platform to enhance usability.
- Clean and structured layout for easy navigation and access to features.
- Subtle animations to provide feedback and guide users through the platform.
